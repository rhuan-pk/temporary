```bash
sudo wget -qP /usr/local/bin/ 'https://raw.githubusercontent.com/rhuan-pk/temporary/master/git_rebase/git_rebase' && chmod -v +x /usr/local/bin/git_rebase && newgrp ${USER:-$(whoami)} && echo -e "script: every \e[1mok\e[m!"
```

## usage

simply type:

```bash
git_rebase
```
